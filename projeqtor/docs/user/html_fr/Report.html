Rapports¶

La liste des rapports sont disponibles par catégorie.

  1. Sélectionnez une catégorie , la liste des rapports sera mise à jour.

  2. Sélectionnez un rapport dans la liste et ses parameters seront affichés.

  3. Modifiez les paramètres pour obtenir l’information désirée.

  4. Cliquer sur un des bouttons pour produire le rapport.

Rapports

Boutons

  • Cliquer sur pour afficher le rapport.

  • Cliquer sur pour obtenir une version imprimable du rapport.

  • Cliquer sur pour exporter le rapport dans en format PDF.

  • Cliquer sur pour l’ajouter dans l’écran “Aujourd’hui”.

Agenda¶

Permet l’affichage des tâches planifiées sur un calendrier pour une ressource.

Calendrier mensuel, hebdomadaire ou quotidien.

Cliquer sur une tâche pour y acceder directement.

Note

Survole de la tâche

  • Vous pouvez voir une courte information sur la tâche.

Coloration des journées

  • Jaune: Jour courrant

  • Gris: Jour non travaillé

Sélection du calendrier

Sélection du calendrier

Période

  • Affiche le mois, la semaine ou le jour.

1er jour

  • Permet de sélectionner le calendrier à afficher.

  • Le premier jour du mois ou de la semaine est affiché.

Ressource

  • Permet de sélectionner le calendrier d’une ressource.

Boutons supérieur

  • Permet de changer le mois, la semaine ou le jour.

Boutons à gauche

  • Cliquer sur pour passer au mode d’affichage “semaine”.

  • Cliquer sur pour revenir au dernier mode d’affichage.

Bouton numéro du jour

  • Cliquer sur le numéro du jour pour passer au mode d’affichage “jour”.

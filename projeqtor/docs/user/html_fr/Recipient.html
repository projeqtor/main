Recipient¶

The Recipient is the beneficiary of bill payments.

Recipients are mainly defined to store billing information.

Other sections

  • Section: Historique des modifications

Section: Description

Field Description
Id Unique Id for the recipient.
Name Short name of the recipient.
Company number Company number, to be displayed on bill.
Tax number Tax reference number, to be displayed on bill.
Tax free Flag to indicate that tax is automatically set to zero for this recipient.
Closed Flag to indicate that recipient is archived.

* Required field

Section: International Bank Account Number (IBAN)

Field Description
Bank Bank name.
Country Country code. IBAN format.
Key Key code. IBAN format.
Account number Full account number defining the BBAN account code.

Field: Key

  • Automatically calculated from other IBAN fields.

Field: Account number

  • Format depends on country.

Section: Addreses

  • Full address of the recipient.

Technical (3/3)
Easy to parameter
•Every user parameter, every list of values may be changed through a devoted screen.
•Default parameters are proposed, corresponding to most common needs.
•Language selection, proposed on locale value, editable by user (only English and French for current version).
Easy to customize
•As Project'Or RIA is proposed under open source GPL Licensing, you may adapt it to your needs.
•Project'Or RIA has been developed as a Framework, so it is very easy to add an element or add a data or change display for an element.
